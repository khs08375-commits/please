
# Lotto Number Generator

## Overview

A simple web application to generate random lottery numbers.

## Project Outline

*   **index.html**: Main HTML structure.
*   **style.css**: Styles for the application.
*   **main.js**: JavaScript logic for generating numbers and handling user interaction.
*   **lotto-ball.js**: A web component to display a single lottery ball.

## Current Plan

1.  **Create `blueprint.md`**: Document the project plan.
2.  **Modify `index.html`**: Set up the basic structure for the lottery number generator.
3.  **Modify `style.css`**: Add styles for a clean and modern look.
4.  **Create `lotto-ball.js`**: Create a web component for displaying a single lottery number.
5.  **Modify `main.js`**: Implement the lottery number generation logic and use the `lotto-ball` component to display the numbers.
